# Budget API

API REST de gestion de budget personnel, construit avec Node.js et MongoDB. Elle gère les enveloppes budgétaires, les dépenses, les revenus, les transactions avec un moyen de faire rappeler les retours d'argent.

## Fonctionnalités 

- **Enveloppes budgétaires** - création et suivi de budgets mensuels par catégorie (logements, courses, loisirs ...)
- **Enveloppes budgétaires** - enregistrement et consultation des dépenses avec génération de rapports par mois PDF
- **Transactions et Retours d'argent** - création d'un formulation pour indiquer qui tu as preté l'argent et quelle est le délai en temps réel avec une notification par mail pour avoir la date de retour
- **Administration** - tableau de bord indiquant le budget du mois, le reste, le total de dépenses et un pourcentage 
- **Export Excel** - export des données budgétaires au format Excel

## Stack Technique 

| Technologie | Rôle |
| React avec TypeScript | Création des différentes pages |
| NodeJS | Création de backend |
| MongoDB | Base des données |
| JWT | Authentification |
| Docker | Containerisation |
| Swagger | Documentation API |
